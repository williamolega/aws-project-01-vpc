# AWS Project 01 - Multi-tier VPC with Secure Segmentation

Production-style VPC with public, private, and isolated subnet tiers across multiple AZs, built entirely with Terraform. First project in an AWS portfolio series translating on-prem networking concepts to cloud.

## Structure

- `architecture/` - diagrams, CIDR plan, tier and flow specification
- `planning/` -phase plan and decision log
- `build-guides/` - one document per build phase
- `validation/` -segmentation matrix and test procedures
- `evidence/` - terminal captures, screenshots, query results
- `terraform/` - infrastructure as code
